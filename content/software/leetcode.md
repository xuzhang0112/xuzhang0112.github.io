## 堆

### 数组中的第k个最大元素

如题

首先实现一个快速选择算法（在left到right内随机选择一个元素，将<该元素和>该元素的分别放到左右两边，并返回该元素的索引），然后多次执行快速选择算法，若选到了n-k，则返回，没选到则相应收缩选取范围

快选：首先随机选取p，放到最左侧，然后i和j从两端向中间移动，若遇到nums[i]> p和nums[j]< p，则交换两个元素，i和j都下一位，循环直至i>=j，全部完成后，p和nums[j]再换回来

### 前k个高频元素

返回数组中频率前k高的元素

先用Counter统计，然后遍历并加入bucket，bucket长度为频率最大值，遍历Counter把所有频率的数字以此放入bucket，然后倒转bucket，取前k个元素

### 数据流的中位数

实现数据结构，支持O(1)查找中位数

用左右两个堆，最大堆存放<中位数的数，最小堆存放>中位数的数；每次加入新数字，若左右两堆数目相同，则中位数放左堆，即从右堆置换出最小值(heappushpop)放左堆(heappush_max)，反之则从左堆置换出最大值(heappushpop_max)放入右堆(heappush)；取中位数时，取左堆的最大值或者左最大和右最小的均值

## 动态规划

### 爬楼梯

每次爬1或2个台阶，多少种方法爬n阶？

初始化长度为n+1的数组，第0阶和第1阶设为1种，从第2阶开始满足dp[i] = dp[i-1] + dp[i-2]

### 杨辉三角

给定numRows，返回杨辉三角的前numRows行

i >= 2, 1 <= j <= numRows时，row[j] = last_row[j-1] + last_row[j]

### 打家劫舍

给定数组，不能连续取，求最多能取到的元素总和

dp长度为n，dp[0] = nums[0], dp[1] = max(nums[0],nums[1])，状态转移方程dp[i] = max(dp[i-1], dp[i-2]+nums[i])

### 完全平方数

给定n，返回和为n的完全平方数的最小数量

对于一切i，满足dp[i] = min(dp[i-j*j]+1, dp[i])；具体来说，完全平方数的dp=1，非完全平方数=减去最大的完全平方数 + 1;

初始化时记录长度为n+1的数组，确保dp[0]=0，其他初始化为inf，遍历时，每次都需要考虑所有的完全平方数，即while j*j<=i

### 零钱兑换

给定数组coins和amount，求组成amount的最小硬币个数

核心是dp[0]=0和dp[i] = min(dp[i],dp[i-coin]+1)，即是否选取这个coin以刷新最小硬币数；遍历1到amount，遍历所有coin；顺序遍历保证了同一coin选取多次

### 单词拆分

给定字符串s和字符串列表wordDict，判断s是否可以有wordDict拼接而成

划分型dp，划分成多个转化为划分成两个；定义dp[i]表示前i个单词是否可以拼接成功，则dp[i] = dp[j] + (s[j:i] in wordDict)，只要任意j满足既可break；注意dp[0]=True以及i从1开始

### 最长递增子序列

数组nums中最长递增子序列的长度

对于任一nums[j]<nums[i]，尝试选取j和i两个元素，刷新dp[i] = max[dp[i], dp[j]+1]；初始化长度为n，值为1，每次考虑i时遍历0到i。

### 乘积最大子数组

数组nums的元素能组成的最大乘积

维护两个数组记录每个位置可以到达的最大值和最小值；遍历，每次刷新最大值和最小值，考虑max(nums[i],nums[i]mx[i-1],nums[i]mn[i-1])，同时刷新ans最大值

### 分割等和子集

将数组nums分成两个子集，和相等

转化为是否可以有子集达到target/2；首先计算target=sum(nums)//2，不能整除直接False；其次，遍历nums，维护dp[i]表示和为i是否可以达到，具体来说，每次遇到一个新的nums[i]，刷新nums[i]到target之间所有dp[j]的状态；最终返回dp[target]



## 多维动态规划

### 不同路径

机器人在m×n的网格从左上到右下的路径数

初始化dp为m×n数组，第一行和第一列每次加1，然后使用递推式```dp[i][j]=dp[i][j-1]+dp[i-1][j]```，返回```dp[m-1][n-1]```

### 最小路径和

在m×n的网格从左上到右下的和最小的路径

类似地，初始化第0行和第0列，然后递推式```dp[i][j]=min(dp[i-1][j],dp[i][j-1])+nums[i][j]```，返回```dp[m-1][n-1]```

### 最长回文子串

返回字符串s的最长回文子串

定义```dp[i][j]```为i到j的子串是否是回文串，初始状态是单个元素是回文串，遍历所有可能的回文串长度，判断所有的i和j是否满足回文串，对于s[i]==s[j]且长度>=3的子串，有递推式```dp[i][j]=dp[i+1][j-1]```，若满足回文，则更新最大子串s[i:j+1]

### 最长公共子序列

text1和text2的最长公共**子序列**，不存在返回0

定义```dp[i][j]```为text1前i个元素和text2前j个元素的最长子序列，则有```dp[i][j]=dp[i-1][j-1]+1 if text1[i-1]==text2[j-1] else max(dp[i-1][j],dp[i][j-1])```,返回```dp[m][n]```

### 编辑距离

word1到word2的最小操作数（插入、删除、替换）

定义```dp[i][j]```为word1前i个元素到word2前j个元素的最小操作数，首先定义初状态即i或j为0时等于单词个数，然后得递推式```dp[i][j]=min(dp[i][j-1]+1，dp[i-1][j]+1，_)，其中_=dp[i-1][j-1]+1 if word1[i-1]!=word2[j-1] else dp[i-1][j-1]`

## 技巧

### 只出现一次的数字

找出数组中只出现了一次的数字

对数组元素迭代做异或运算reduce(xor,nums)（异或的性质是相同元素做异或得到0，不同元素做异或得到本身，满足交换律）

### 多数元素

找出数组中的众数

用ans和hp分别存储众数和它的频次，每次遇到则hp+1，遇到其它数字则hp-1，如果hp=0则更新众数为当前数字，频率设为1；最终返回ans

### 颜色分类

数组中012分别表示红白蓝，排序按照红白蓝排

用p1存储0或1的个数，p0存储0的个数，遍历 ，默认设为2，遇到0或1则将p1位置设1，p1+1；遇到0则p0位置设0，p0+1；

### 下一个排列

找到数组下一个排列，对应更大的数字

首先从后往前找升序(i,j)；其次从后往前找nums[k]>nums[i]，交换（因为是降序，所以是第一个>nums[i]的数，增幅最小）；最后将i之后的元素变为升序（因为是降序，头尾交换既可）

### 寻找重复数

寻找数组中重复的数字

用数组元素表示下一个访问的位置索引，类比链表，用快慢指针找到环的入口；首先用快慢指针，使之在环内相遇；其次用一个新的指针finder，必然和慢指针相遇在环的入口，即重复数（证明：快慢指针，推得慢指针走过的距离是环的整数倍，因此，起点到环的距离+环内现在的距离=环长；进一步，Finder和slow都走同样的距离既可）
