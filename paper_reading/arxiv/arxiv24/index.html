A Novel ICD Coding Framework Based on Associated and Hierarchical Code Description Distillation

Xu Zhang | Suggest Changes

原文链接

ICD编码任务如上图所示，根据医学文本，分类出对应的ICD code

每个医学报告对应若干ICD code，通过先验知识ICD Tree，可以获得若干相邻的（易混淆的）ICD code。

监督信号来自两方面：

  1. 文本描述和正确的标签描述都应该能被分类器正确地分类出icd code；

  2. 文本描述应该和标签描述尽可能接近，和易混淆的标签描述尽可能远离；

对于不同的Backbone，该方法在MIMIC III上可以产生0-3个点的提升。
